# HR Attrition Analysis Dashboard (Excel)

## 📌 Project Overview

This project is an interactive HR Attrition Dashboard built in Microsoft Excel using the IBM HR Analytics dataset.

The dashboard helps analyze employee attrition by department, gender, age group, and job role using PivotTables, PivotCharts, KPI cards, and slicers.
